## Sobre este repositorio

Este repositorio recoge el material, los ejercicios y las prácticas del módulo de **NLP** del bootcamp de [KeepCoding](https://keepcoding.io/).

Aquí encontrarás:

- 📊 **Datasets** — conjuntos de datos utilizados durante las sesiones.
- 💡 **Soluciones** — resoluciones propuestas a los ejercicios.
- 🖼️ **Slides** — diapositivas de apoyo.
- 📄 **Practica_final_NLP.pdf** — enunciado de la práctica final.
